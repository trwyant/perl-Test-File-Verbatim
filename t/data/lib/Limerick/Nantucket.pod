=head1 NAME

Limerick - Grist for the Test::File::Verbatim mill

=head1 SYNOPSIS

 use Limerick::Nantucket;
 print Limerick::Nantucket->limerick();

=head1 DESCRIPTION

This unsupported module serves no useful purpose in and of itself. It
exists only to support testing of
L<Test::File::Verbatim|Test::File::Verbatim>, and may be changed or
revoked without notice.

=head1 METHODS

This class provides the following B<unsupported> methods:

=head2 limerick

This static method returns a limerick the author of this package finds
amusing. At the moment, this is:

=for VERBATIM CONFIGURE trim on
=for VERBATIM BEGIN Limerick::Nantucket
    There was an old man from Nantucket
    Who kept all his cash in a bucket.
	But his daughter named Nan
	Ran away with a man,
    And as for the bucket, Nan tuck it.
=for VERBATIM END
